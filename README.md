# ChattApp

ChattApp is a real-time messaging web application built with the MERN stack, designed to provide a seamless chat experience. This project helped me deepen my understanding of web development, MERN stack practices, and deployment-level considerations.

## Features

Real-time Messaging: Send and receive messages instantly image sharing also possible.

Welcome Email: Users receive a welcome email upon signup or login.

User Authentication: Secure login and signup system.

Responsive UI: Built with Tailwind CSS and DaisyUI for a modern and responsive design.

State Management: Implemented using Zustand for efficient state handling.

Response Count Tracking: Arcjet is used for handling response counts.

Cloud Storage: Integrated with Cloudinary for media handling.

## Installation

Clone the repository:

git clone <repo-url>


Install backend dependencies:

npm install --prefix backend


Install frontend dependencies:

npm install --prefix frontend


Start the development server:

npm run start
